# GitHub এ push করার জন্য commands:
# প্রথমে GitHub এ repository তৈরি করুন, তারপর এই commands চালান:

# 1. Remote repository যোগ করুন (আপনার URL দিয়ে replace করুন)
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git

# 2. Code push করুন
git push -u origin master

# 3. ভবিষ্যতে changes push করতে:
git add .
git commit -m "Your commit message"
git push
